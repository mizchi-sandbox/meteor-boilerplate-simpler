# meteor-boilerplate-simpler

written by coffeescript by @mizchi

## How to install

```
# clone or fork this
mrt install
meteor
```

## Run tests

```
npm install -g laika
./test
```

### Packages used

* semantic-ui
* collection2
* less
* jquery
* underscore
* handlebar-helpers
* iron-router
* iron-router-progress
* accounts-password
* accounts-ui
* coffeescript
* typescript-compiler
